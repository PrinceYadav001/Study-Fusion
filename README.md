# 📚 Study Fusion – Galgotias University B.Tech Resources

A free, student-built platform providing organized study resources for all 8 semesters of B.Tech at Galgotias University. Includes YouTube video links, notes, and subject-wise content — all in one place.

> 🎓 Made by **Prince Yadav** | [YouTube Channel](https://www.youtube.com/@Prince_Yadav-z8k)

---

## 🌐 Live Preview

**[https://studyfusion.vercel.app/](https://studyfusion.vercel.app/)**

---

## 📁 Project Structure

```
Study-Fusion/
│
├── index.html          # Home page (semester selection + events)
├── styles.css          # Main stylesheet (all root-level styles)
├── main.js             # JavaScript (scroll animations, copyright year)
│
├── sem1/               # Semester 1 resources
│   ├── semester1.html  # Subject cards for Sem 1
│   ├── subject1.html   # Individual subject page (videos + notes)
│   ├── style.css       # Semester-level stylesheet
│   └── sem-main.js     # Semester JavaScript
│
├── sem2/ ... sem8/     # Same structure repeated for Semesters 2–8
│
└── Assests/
    └── Batches/        # Semester card images + logo
```

---

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| **HTML5** | Page structure and content |
| **CSS3** | Styling, layout (Flexbox), animations |
| **Vanilla JavaScript** | Scroll-reveal, auto year, ripple effect |
| **Google Fonts (Poppins)** | Clean modern typography |
| **Bootstrap 5** | Button utilities on home page |
| **YouTube Embed** | Video resources inside subject pages |

> ✅ **No frameworks. No npm. No build tools.** — Just open and run!

---

## ✨ Features

- 🎯 **8 Semester Cards** — Direct links to subject resources for each semester
- 📺 **Embedded YouTube Videos** — Chapter-wise video playlists per subject
- 📝 **Notes Download Links** — PDF notes organized by chapter
- 🎪 **Upcoming Events Section** — Hackathons and activities
- 🖱️ **Scroll Animations** — Cards fade in as you scroll (Intersection Observer)
- 🏠 **Clickable Logo** — Logo on every page navigates back to home
- 📱 **Responsive Design** — Works on mobile, tablet, and desktop

## 🚀 How to Run

1. **Download or clone** this repository
2. **Open** `index.html` in your browser (Chrome/Firefox/Edge)
